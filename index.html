What is JavaScript?

Website = HTML + CSS + JavaScript

JavaScript :
  • Scripting Language that is used to render dynamic content on webpages.
  • Embedded in your HTML page and interpreted by Web Browsers.
  • Its functionality can be extended to Object Oriented Programming.
  • External JavaScript files have '.js' extension.
  • JavaScript ≠ Java

#1 Your First Script!

<!DOCTYPE HTML> <html> <head> <title>My First Script</title> </head> <body> <script type="text/javascript"> alert('Hello World!'); </script> </body> </html>
  • JavaScript code gets executed when the webpage loads or an event is triggered.
  • You can place any number of script tags in a HTML document.
  • However, only 1 block of JavaScript code can be executed at a time.

#2 Let's display it ...

<!DOCTYPE HTML> <html> <head> <title>JavaScript</title> </head> <body> <script type="text/javascript"> document.write('I am learning JavaScript!'); </script> </body> </html>
  • JavaScript does not have any print or output functions.
  • It is used to manipulate HTML documents.

#3 Rendering HTML within a Document

<!DOCTYPE HTML> <html> <head> <title>JavaScript</title> </head> <body> <script type="text/javascript"> document.write('<b>This should be bold</b>'); document.write('This is normal text'); </script> </body> </html>

#4 Image OnClick()

<!DOCTYPE HTML> <html> <head> <title>OnClick</title> </head> <body> <img src="assets/img/gdg-bangalore.png" onclick="alert('You clicked on the Image!')" /> <script type="text/javascript"> </script> </body> </html> <body> <img src="assets/img/gdg-bangalore.png" onclick="func_name('You clicked on the Image!')" /> <script type="text/javascript"> function func_name(message) { alert(message);} </script> </body>

JavaScript Objects

  • JavaScript Objects are variables with Properties & Methods.
  • Properties are values associated with the object.
  • Methods are actions that the objects can perform.

Example : Car

  • Properties
car.name = Ford; car.model = Figo; car.color = Black;
  • Methods
car.start(); car.drive(); car.brake(); car.stop();

JavaScript Objects

  • Almost everything in JavaScript is treated as an Object.
var person = { firstName : "Kartik", lastName : "Kannapur", age : 22, org : "GDG Bangalore", fullName : function() {return person.firstName + " " + person.lastName} };
  • Chrome : JavaScript Console
  • Accessing Object Properties
> person; > person.age;
  • Accessing Object Methods
> person.fullName();
